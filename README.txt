INTRODUCTION
------------

Around Media lets you embed Around Media tours into your site or application's
content. A field type, widget and formatter are provided.

INSTALLATION
------------

Install as you would normally install a contributed Drupal module. See
https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules.

CONFIGURATION
-------------

 * Add an Around Media field to an existing content or entity type.
 * When creating field contents, fill in the Around Media tour code.
 * Display the content. The field will show the chosen Around Media tour.
